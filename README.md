# 팀원 공유사항

## 요구사항 명세서
- 팀이 직접 분석·정의한 요구사항 + 검증 기준 + 시연 타임스탬프

## 결과보고서
- 프로젝트 내용·설계·결과 정리 (PDF·PPT 두 가지 형식 모두 제출)

## 결과파일
- **result_file -> 추후 SW_Bootcamp_13기_X반_X팀_결과파일.zip 형식으로 제출**
- 소스코드·테스트·실행 자원·Git 히스토리

## 시연영상
- 요구사항별 동작 시연

## 포스터
- 프로젝트 요약 (PDF·PPT 두 가지 형식 모두 제출)

## 평가 결과 리포트
- 본 평가 Agent를 직접 실행해 생성한 자체 평가 결과

## 제출 전 체크리스트
- [ ] 6종 산출물(명세서·결과보고서·결과파일·시연영상·포스터·평가결과)을 모두 제출했다
- [ ] 결과보고서와 포스터를 PDF·PPT 두 가지 형식으로 모두 제출했다
- [ ] 본 평가 Agent를 직접 실행해 생성한 ..._평가결과.html을 포함했다
- [ ] 파일명이 형식(SW_Bootcamp_13기_X반_X팀_요구사항_명세서.md 등)에 맞는다
- [ ] 요구사항 명세서에 요구사항·검증 기준·시연 타임스탬프가 채워져 있다
- [ ] 시연 영상에서 각 요구사항 동작이 타임스탬프 지점에 나온다 (불가 항목은 N/A 표기)
- [ ] 결과파일.zip에 소스·테스트·실행 문서·Git 히스토리가 들어 있다
- [ ] RUN.md가 DevOps 8개 기준을 충족한다 (가이드대로 따라 하면 실행된다)
- [ ] 테스트가 자동 실행되거나, 실행 결과가 첨부되어 있다
