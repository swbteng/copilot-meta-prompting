### 1. `refine_copilot.py` 재구조화
- **OBSERVE 모드/`MODE` 개념 완전 제거.** 프록시는 이제 항상 교체 동작.

### 2. 로깅 재설계 (관측 → 감사 로그)
- `LOG_ENABLED` 토글(기본 on)로 **파일 로깅 온/오프**. 콘솔 교체 알림은 항상 표시.
- 교체 1건마다 **`observe-YYYY-MM-DD.log`에 JSONL 한 줄** append. 날짜가 바뀌면 자동으로 새 파일.
- 레코드: `{ts, channel, model, before, after}`. **추후 VS Code 확장이 그대로 파싱할 데이터 계약.**

### 3. 정제 서버 호출 구조 + 고정 문자열

  현재는 **고정 문자열 `"안녕,클로드"`** 를 반환하도록 함 → 서버 연동 전 파이프라인 검증 가능.
- 서버 호출은 `call_refine_api()`(stdlib `urllib`, 캐시, fail-open). 엔드포인트는 공란.
  확정 시 `(1)요청`/`(2)응답` 두 군데만 맞추면 됨.
