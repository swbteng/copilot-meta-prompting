# nested extension 폴더 안내

이 폴더는 개발 과정에서 남아 있는 중첩 확장 산출물이다. 최종 제출 기준의 활성 확장 구현과 실행 문서는 상위 폴더 `src/program_extension/README.md`와 `src/program_extension/package.json`을 기준으로 한다.

평가 및 재현 시 우선 확인할 경로:

```text
result_file/src/program_extension/
├── package.json
├── src/extension.js
├── src/chat.js
├── src/refiner.js
├── media/
└── build.sh
```

이 README는 결과파일 내 모든 README가 평가자에게 역할을 설명하도록 남긴 안내 문서이다.
