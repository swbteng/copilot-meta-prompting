9. 확인 불가 / 한계 항목

  • 확인 불가 AI 통합 테스트는 실제 LLM/임베딩/리랭커 서버가 없는 환경이므로 skip 처리됨. 근거: result_file/tests/program_AI/test_serving.py 및 실제 pytest 실행 결과.
  • 확인 불가 시연 영상 타임스탬프는 요구사항서상 미정으로 표기되어 있으며, 실제 영상 내 타임스탬프는 본 환경에서 확인되지 않음.
  • 확인 불가 VS Code Extension Development Host에서의 실제 UI 상호작용은 본 자동 검토 환경에서 직접 실행하지 못했으나, Node 기반 테스트는 통과함.
